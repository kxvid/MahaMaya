## 1. NAME

### The word
**Kahla** — pronounced `KAH-la` (rhymes with "lava," not "Kayla").

### Etymology
Direct phonetic restyling of the Sanskrit word *Kala* (काल), meaning:
- Time
- The opportune moment
- Transformation / the agent of change

The **H** is inserted to force the correct vowel sound for English-speaking clients. "Kala" alone reads ambiguously as "KAY-la" (the name Kayla). "Kahla" removes the ambiguity.

### Why the name survived ten rejected alternatives
The naming process tested four categories before landing here:

| Category | Tested | Why they failed |
|----------|--------|-----------------|
| Greek/Latin philosophical | Apeiron, Noema, Aureus, Kairo | 8+ digital agencies per keyword. Unwinnable SEO. |
| Sanskrit-derived | Aditara, Vasuria | Either .com taken by Indian retailers or read as foreign to the target market. |
| Stylized misspellings | Kaden, Rizen, Embr | All claimed by direct competitors in digital/brand agency niche. |
| Neutral English | Meridian | City name. Hundreds of geographic agency collisions. |

**Kahla** wins because:
1. Different category competition (photography studio in Germany, beauty salon in AZ — zero digital agency conflicts)
2. Phonetically neutral to white LA clients
3. Personally meaningful (Kala / Mahakala is the founder's Hindu devotional through-line)
4. Numerologically aligned with the founder's chart (see Section 2)

## 4. VISUAL SYSTEM

### Color palette

| Role | Hex | Name | Usage |
|------|-----|------|-------|
| Page background | `#080705` | Void | Primary dark surface |
| Card / panel | `#131110` | Charcoal | Elevated surfaces |
| Surface | `#1C1A17` | Surface | Layered surfaces |
| Raised | `#252118` | Raised | Hover, active states |
| Primary accent | `#C9A34F` | Gold | CTAs, key emphasis |
| Accent light | `#E2C27A` | Gold Light | Hover on gold |
| Secondary accent | `#9B4E30` | Terra | Secondary callouts, warnings |
| Primary text | `#F0E6CC` | Cream | All body copy |
| Muted text | `#8A8070` | Muted | Captions, meta |
| Faint text | `#4A4438` | Faint | Disabled, ultra-subtle |

### Rules
- Never use pure `#000000` or pure `#FFFFFF`. Always use `#080705` and `#F0E6CC`.
- Gold (`#C9A34F`) is for emphasis only — one focal point per section, not decoration.
- Terra is used sparingly (error states, destructive actions, secondary callouts).
- Dark is the default. Light-mode variants exist but are not the primary visual identity.

## 5. TYPOGRAPHY

### Font pairing
- **Display / Headlines:** Cormorant Garamond, weight 300
- **Body / UI:** DM Sans, weight 300

### Scale
| Style | Font | Weight | Size | Use |
|-------|------|--------|------|-----|
| Hero | Cormorant Garamond | 300 | 64px | Landing page headline |
| H1 | Cormorant Garamond | 300 | 42px | Page titles |
| H2 | Cormorant Garamond | 400 | 28px | Section titles |
| H3 / Label | DM Sans | 500 | 16px uppercase, tracked 0.12em | UI labels |
| Body | DM Sans | 300 | 15px, line-height 1.75 | All paragraph text |
| Caption | DM Sans | 300 | 12px uppercase, tracked 0.08em | Meta, tags |
| Gold italic | Cormorant Garamond | 400 italic | 20px | Pull quotes, accent lines |

### Rules
- Sentence case always. Never Title Case. Never ALL CAPS except in small caption labels (tracked).
- Body copy is always DM Sans 300. Never bold the body.
- Headlines are always Cormorant Garamond 300. The ultra-light weight is the brand signature.
- No other fonts. No Inter. No Space Grotesk. No Arial.

- Encodes the Sarvatobhadra Chakra — the 8-directional Vedic chart geometry from the founder's natal chart.
- Outer ring and inner ring represent the 12 rashis and 27 nakshatras.
- Eight directional points align with the Ashtadikpalas (directional regents).

- K-letterform integrated into inner ring.

### External reading
Clients see a premium geometric compass mark. Reads as strategy, navigation, precision. Architecture-firm adjacent. No religious or cultural signaling visible at surface level.

### Logo variants
1. **Mark only** — 8-direction compass + K, used as favicon and avatar
2. **Wordmark only** — KAHLA set in Cormorant Garamond 300, tracked 0.3em
3. **Horizontal lockup** — mark + wordmark + optional "STUDIO" descriptor
4. **Light surface variant** — inverted for use on cream backgrounds

### Construction
- Mark is 80x80px on a 4px grid
- All strokes 0.5px at reference size
- Gold color only. Never color-shifted except to cream on dark, or void on light.

## 7. VOICE

### Three-word voice
Confident. Surgical. Economical.

### Write like this
- "We don't run ads. We build systems that compound."
- "Your booking rate goes up. Your no-shows go down."
- "Kahla: the opportune moment. That's what we find for your business, and build around."

### Don't write like this
- "We're a passionate team dedicated to transforming your digital presence."
- "Hey! We help small businesses grow online using cool AI tools lol"
- "We create innovative solutions for businesses of all sizes."

### Rules
- No em-dashes. Use periods or commas.
- No hedging language. No "might," "perhaps," "arguably" in client copy.
- No AI-sounding phrases: "delve," "navigate the landscape," "in today's fast-paced world," "cutting-edge solutions."
- Short sentences. Declarative. Every sentence earns its place.
- Read everything out loud. If it sounds like marketing copy, rewrite it.

## 10. USAGE RULES

1. **Always use Cormorant Garamond 300 for headlines.** Non-negotiable.
2. **Never use pure white or pure black.** Use Cream (#F0E6CC) and Void (#080705).
3. **Gold is for emphasis, not decoration.** One focal point per section.
4. **Never place the logo over photography or gradients** without a solid overlay.
5. **Spacing is the brand.** Generous whitespace, 0.5px borders, minimal decoration.
6. **No rounded corners above 4px.** Structured and architectural, not SaaS-soft.
7. **Sentence case everywhere.** No Title Case, no ALL CAPS except tracked captions.
8. **No emojis in client-facing copy.** Ever.
9. **No stock photography.** Either original photography, abstract geometric imagery, or no imagery.

## 11. FOR CLAUDE CODE USAGE

When building components, pages, or content for Kahla Studio in Claude Code, reference this file with:

```
@BRAND.md
```

Default assumptions Claude Code should make:
- Dark theme first, light theme as alternate
- Typography: Cormorant Garamond display, DM Sans body
- All colors from the palette in Section 4, addressed by CSS variables
- Voice per Section 7 — short, declarative, no marketing fluff
- Mobile-first, but design for desktop at 1440px as the reference

Component naming convention:
- Components: PascalCase (`HeroSection`, `ServiceCard`)
- Files: kebab-case (`hero-section.tsx`)
- CSS variables: kebab-case with `--k-` prefix (`--k-gold`, `--k-cream`)
- Utility classes: `k-` prefix (`k-btn-primary`, `k-card`)
